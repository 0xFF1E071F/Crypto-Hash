RC4 Test Suite verifies the correctness of RC4/Alleged-RC4 algorithm implementation.

RC4_nessie.exe	- RC4 test vector generator, as specified by Project NESSIE.
RC4_test.exe	- RC4 test as specified in 'A Stream Cipher Encryption Algorithm
		  "Arcfour"' (Kaukonen, Thayer) and 802.11 RC4 Test Vector Generator
		  (Johnston)

All tests are compiled with aRC4 Library version 1.0
[SHA1(rc4.lib)		= 3AA3 3382 8886 9774 5410  97A2 4AFB FB54 A88F C42B].
It also uses utils.lib
[SHA1(utils.lib)	= 6CC7 9E61 0CF1 E0FF 64C2  8B7C C9C4 13A0 A1FB 25F3].
